**Overview of Annex A- ISO/IEC 27001:2022
• Annex A of ISO/IEC 27001:2022 provides a reference list of 93 information security controls that organizations can use to treat risks identified in their risk assessment. 
• These controls are not mandatory but serve as guidance to ensure organizations have considered a comprehensive set of options to manage their information security risks. 
• Purpose: To provide a control set aligned with modern threats and support the implementation of an Information Security Management System (ISMS). 
• Alignment with ISO/IEC 27002:2022: All control in Annex A align with ISO/IEC 27002.
• Organizations must compare their selected controls against Annex A (a process called the Statement of Applicability - SoA) to justify inclusions or exclusions.

**Overview of Annex A - ISO/IEC 27001:2022**
• Total Controls: (93) controls. 
• Grouped into (4) Themes (Domains): 
    • Organizational Controls (Clause A.5) - 37 controls Policies, roles, responsibilities, project management, supplier relationships, etc. 
    • People Controls (Clause A.6) - 8 controls Awareness, training, disciplinary actions, screening, etc. 
    • Physical Controls (Clause A.7) - 14 controls Physical access, equipment security, protection against environmental threats, etc. 
    • Technological Controls (Clause A.8) - 34 controls Access control, cryptography, logging, malware protection, backup, etc.

**Overview of ISO/IEC 27002:2022** 
• Guidance document that provides implementation advice for the controls listed in ISO/IEC 27001 Annex A 
• Focus: Best practices and detailed security controls 
    • Includes: 
        • 4 themes: Organizational, People, Physical, Technological
        • 93 controls with attributes (Purpose, Control Type, Operational Capabilities, etc.) 
        • detailed explanation for each control, as follows 
            • Purpose 
            • Implementation guidance 
            • Control attributes (e.g. control type, threat addressed)
