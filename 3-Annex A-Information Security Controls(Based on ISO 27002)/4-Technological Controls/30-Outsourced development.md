
**Control** 
    The organization should direct, monitor and review the activities related to outsourced system development. 


![](../../PNGs/Pasted%20image%2020260203181315.png)

**Purpose** 
    To ensure information security measures required by the organization are implemented in outsourced system development.

**Guidance (Summary)** 
- Define security requirements in contracts. 
- Monitor and review outsourced work.
- Protect intellectual property and source code.

**Other Information** 
    Use NDAs and secure communication channels.

**Difficulties in Control Interpretation and Application**
- Contracts often lack detailed security expectations or control clauses. 
- Oversight and review of third-party development work is minimal.
- Source code protection and IP control mechanisms are weak or missing.
