
**Control** 
    Security measures should be implemented when personnel are working remotely to protect information accessed, processed or stored outside the organization's premises.

![](../../PNGs/Pasted%20image%2020260203112935.png)


**Purpose** 
    To ensure the security of information when personnel are working remotely.

**Guidance (Summary)** 
- Define remote work security policies and responsibilities. 
- Use secure connections, encryption and access controls. 
- Train staff on secure remote working practices

**Other Information** 
    Consider physical and technical risks.

**Difficulties in Control Interpretation and Application** 
- Organizations struggle to enforce uniform security controls across diverse remote setups. 
- Physical security risks (e.g., shared home devices) are often ignored. 
- Staff may lack training on secure remote access and device use. ...':
